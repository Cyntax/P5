P5 Framework

P5 not only provides a base framework for beginner's, it also explains, it the simplest possible way, some of the key programming concepts that beginner's usually overlook. Following are some of the concepts that P5 is build upon, so as to help a beginner understand them better and be able to work through the code easily while reading it.

  • Functions Returning Proper Values
  • Avoiding Dependency Injection
  • Single Responsibility Principle
  • Layered Functionality Logic
  • OOP: Inheritance and Overriding
  • OOP: Regular and Static Methods
  • OOP: Code Reusability
  • PHP5: Exception Handling
  • PHP5: Autoloading Classes
  • PHP5: Anonymous (throw away) Functions
